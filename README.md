# CodeAlpha Music Player

## 📌 Project Overview
This is a modern and interactive Music Player built using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## 🚀 Features
- Play and Pause functionality
- Next and Previous song navigation
- Dynamic song title and album cover display
- Progress bar with seek control
- Volume control with icon
- Auto-play next song
- Smooth and clean user interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📷 Preview
A stylish music player with interactive controls and responsive design.

## 📎 How to Run
1. Download or clone the repository
2. Add your music files (MP3) and cover images
3. Open `index.html` in your browser
